# 2. Use GOV.UK Notify for email notifications

Date: 2026-06-16

## Status

✅ Accepted

## Context

Customers may request email notifications for file receipt, file processing completion, or any other reason we need to send them an email.

## Decision

We will use GOV.UK Notify for sending email notifications.

## Consequences

Using GOV.UK Notify will allow us to send email notifications to customers in a secure and reliable manner. It also provides a simple API for integration, which will streamline the development process and reduce the complexity of our notification system. Additionally, GOV.UK Notify is a government-approved service, ensuring compliance with relevant regulations and standards.

We will need to manage our GOV.UK Notify account and ensure that we adhere to best practices for email communication and manage email templates effectively.
